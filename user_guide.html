🧾 KF領収書管理システム

使い方ガイド

📱システム概要

このシステムは、イベントや活動での予算管理と領収書管理を簡単に行えるWebアプリケーションです。スマートフォンやパソコンから誰でも簡単に使用できます。

🚀基本的な使い方の流れ

1イベントを作成する

  1. トップページで「新しいイベント追加」フォームに入力
  2. イベント名: 「春の学園祭」「文化祭」など
  3. 予算: 総予算額を円単位で入力（例：100000）
  4. 「追加」ボタンをクリック

2チームを作成する

  1. 作成したイベントの「選択」ボタンをクリック
  2. チーム一覧画面で「新しいチーム追加」フォームに入力
  3. チーム名: 「実行委員会」「装飾チーム」など
  4. 予算: そのチームに割り当てる予算額
  5. 「追加」ボタンをクリック

3領収書を登録する

  1. チームの「選択」ボタンをクリック
  2. 「新しい領収書追加」フォームに入力
  3. 個人名: 支払いをした人の名前
  4. 用途: 何を買ったか（例：「装飾用品」「食材」）
  5. 金額: 支払った金額を円単位で入力
  6. 「追加」ボタンをクリック

💡便利な機能

📋使用予定用途の登録

実際に買い物をする前に、使用予定を事前に登録できます：

  • 用途: 予定している購入内容
  • 担当者名: 買い物担当者
  • 予定金額: 使用予定の金額
  • 備考: 詳細な内容や注意事項

📊予算管理概要の確認

「予算概要」ボタンから以下を確認できます：

  • イベント全体の予算状況: 総予算、使用済み、残予算
  • チーム別詳細: 各チームの使用状況と使用率
  • 視覚的な表示: 使用率をバーで表示
  • アラート機能: 予算超過の警告

🎯実際の使用例：学園祭の予算管理

  1. イベント作成: 「秋の学園祭2025」予算50万円
  2. チーム作成:
    • 装飾チーム（15万円）
    • 食品販売チーム（20万円）
    • ステージチーム（10万円）
  3. 使用予定登録: 装飾チームが「風船購入」5000円を事前登録
  4. 領収書登録: 実際に風船を購入したら領収書を登録

📱スマートフォンでの使い方

🛒外出先での領収書登録

  1. 買い物後、その場でスマホからシステムにアクセス
  2. すぐに金額と用途を入力して登録
  3. リアルタイムでチームに共有

💰予算確認

  • 買い物前にスマホで残予算を確認
  • 使用予定を登録して予算オーバーを防止

⚠️注意事項とコツ

�領収書登録時のコツ

  • すぐに登録: 買い物後はできるだけ早く登録
  • 用途は具体的に: 「装飾用品」より「風船・リボン」の方が分かりやすい

💡予算管理のコツ

  • 使用予定の活用: 大きな買い物は事前に登録して調整
  • 定期的な確認: 週1回程度は予算概要画面で全体状況を確認

🔔アラート機能

システムが自動的に以下の警告を表示します：

  • 🔴 予算超過: チームの使用率が90%を超えた場合
  • 🟡 予算注意: 使用予定込みで予算オーバーの可能性がある場合

🎯まとめ：誰でも簡単3ステップ

1. 📝登録

イベント→チーム→領収書の順で登録

2. 📊確認

予算概要で使用状況をチェック

3. 🔔共有

リアルタイムでチーム内共有

このシステムを使うことで、複数人での予算管理が透明で効率的になり、予算オーバーを防ぎながらスムーズなイベント運営が可能になります。
